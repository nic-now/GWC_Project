### What it is

- This project is an educational website about DNA enzymes and their function in replication, built with HTML, CSS, and JavaScript. 
- *Key features*: interactive thermal denaturation predictor, base pair counter, complementary strand generator, and enzyme overview pages.



### What we used:
- Frontend: HTML, Javascript, CSS
- Built with Github Pages
