---

### Option 1: VSCode


1. [Download/Install VS Code](https://code.visualstudio.com/)
2. [Install Git](https://git-scm.com/downloads), this is for commands and to push/edit to remote repo.
3. Open VS Code, go to Source Control
4. ...editing

---

### Option 2: GitHub Desktop

1. [Download GitHub Desktop](https://desktop.github.com/).
2. Log in with your GitHub account.
3. Click **Clone a repository**
   ```
   #this is the repo url
   https://github.com/nic-now/GWC_Project.git
   ```

---

### Option 3: Command Line/ Terminal 
[Github Instructions](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
